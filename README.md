# EXEIN

AI-Powered Gym Exercise Tracking and Analytics System

## Overview
EXEIN is a computer vision based fitness tracking system that automatically detects exercises, counts repetitions, tracks workout sessions, and generates fitness analytics reports.

## Features
- Exercise Detection
- Rep Counting
- Set Tracking
- Calorie Estimation
- Weekly Reports
- Monthly Reports
- Member Analytics

## Planned Exercises
- Squats
- Bicep Curls
- Push-ups
- Lunges
- Shoulder Press

## Tech Stack
- Python
- OpenCV
- MediaPipe
- Machine Learning
- Flask / FastAPI
